Date Picker

Used when entering dates on the main form. This is a jQuery plug-in and produces a more user-friendly way to pick a date.

Drag and Drop

This is a feature where a user can drag and drop an image of a device using Javascript. This could be used when adding a courtesy phone.

Get Current Location

This Javascript feature can be used to acquire the user's current location and calculate distance or show the route to the nearest Phone Fix location.

Click the button to get your current location.

Search FAQS

This feature enables the user to search through the FAQs page and get relevant questions and answers based on their input. This uses AJAX with some jQuery to get the data.


Interactive Map

This interactive map would show users the different locations of the Phone Fix shop. They can hover their cursor over the map and see the address of each location.
